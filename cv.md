## Code example:

Gode from solved LeetCode problem:
*Given an integer x, return true if x is a palindrome, and false otherwise.*


```
class Solution:
    def isPalindrome(self, x: int) -> bool:
        start = str(x)
        reversed = start[::-1]
        if reversed == start:
            return True
        return False
```
